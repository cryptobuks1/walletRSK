This project is a **RSK-based web wallet** with login system. Registered users will be able to see their tokens balance token transfers, NFTs and others data. Those features will be realised using the **Covalent API**.

## Features:

- Registration & login system;
- Ability to **link new wallets** to your account and delete unnecessary ones;
- View the average **Gas price**;
- View **RBTC stats** of your wallet: balance, total fees, income and expenses;
- View your **tokens balance** and **transfer history**;
- View your **NFTs**;
- tRBTC token **faucet** with claim history [https://faucet.rsk.co/];
- Market stats with some **token prices** (price data taken from https://live.sovryn.app/stats);
- **Sovryn TVL** [Total Value Locked] stats;
